# 小红书长文编辑器 - 需求文档

## 项目概述
将小红书风格的长文笔记导出为 JPG 图片的工具。

## 功能需求

### 1. 编辑区（中间）
- [x] 多页卡片布局，每页比例 9:16（宽375px，高667px）
- [x] **第一页**：标题输入区 + 正文输入区
- [x] **中间页**：纯正文输入区（内容超出自动分页）
- [x] **最后一页（结尾卡）**：固定样式引流卡片，圆角矩形，带装饰线和文字「职场笔记整理好，放在你手边」
- [x] 页面底部圆形页码导航，当前页红色高亮，点击跳转

### 2. 工具栏（顶部）
- [x] 撤销、重做按钮
- [x] H1、H2 标题切换
- [x] 有序列表、无序列表
- [x] 引用块
- [x] 下划线
- [x] 插入图片（占位）
- [x] 插入 emoji（弹出面板）

### 3. 模板面板（右侧）
- [x] 两个 Tab：「选择模板」和「封面设置」
- [x] 三组模板（点击实时切换）：
  - 简约基础（白色背景，黑色字体）
  - 清晰明朗（米白色背景，温暖风格）
  - 黑白极简（黑色背景，白色字体）
- [x] 结尾卡样式跟随模板同步变化

### 4. 底部状态栏
- [x] 左侧显示「字数：XXX」和「自动保存于 HH:MM」
- [x] 右侧：「暂存离开」（灰色）和「下一步」（红色）

### 5. 导出功能
- [x] 使用 html2canvas 逐页截图
- [x] scale: 2 高清导出
- [x] 文件名格式：「笔记_第1页.jpg」
- [x] 导出进度条显示

### 6. 分页逻辑
- [x] 使用 contenteditable div 实现
- [x] 按 Enter 两次可手动插入分页
- [x] 内容超出时自动检测（待完善）

## 技术栈
- React + TypeScript + Vite
- html2canvas（导出 JPG）

## 样式规范
- 整体 light mode，背景色 #f5f5f5
- 字体：-apple-system, PingFang SC, Microsoft YaHei
- 卡片圆角：16px
- 主题色：#ff2442
